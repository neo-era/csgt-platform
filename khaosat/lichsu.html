Cấu hình URL dữ liệu

Để hiển thị lịch sử thao tác, bạn cần publish sheet LichSu trong Google Sheet thành CSV và điền URL vào bên dưới.

  1. Mở Google Sheet của dự án
  2. Vào File → Share → Publish to web
  3. Chọn sheet: LichSu → định dạng: CSV
  4. Nhấn Publish → copy URL
  5. Dán URL vào ô bên dưới và nhấn Lưu
